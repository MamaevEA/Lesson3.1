# Подсказка по командной строке

Сенить директорию 
Windows
``` sh
cd C:\<folder_name>
```

MAC, Linux
``` sh
cd Users/<folder_name>
```

Очистить терминал
Windows
```sh
cls
```

MAC, Linus
```sh
clear
```

Создать директорию
```sh
mkdir new_folder
```

Листинг директории

Windows
```sh
dir
```

MAC, Linux
```sh
ls
```
